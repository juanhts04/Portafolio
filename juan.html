Conoce a Juan Carlos

Soy un desarrollador web apasionado y versátil con experiencia en la creación de aplicaciones y sitios web dinámicos y funcionales. Mi conjunto de habilidades incluye un dominio completo de HTML, CSS y JavaScript, así como experiencia avanzada en marcos y bibliotecas modernas como React. Además, cuento con sólidos conocimientos en programación backend utilizando Java y Python, lo que me permite abordar proyectos de desarrollo de manera integral y eficiente. También tengo experiencia en manejo y administración de bases de datos utilizando herramientas como HeidiSQL y MySQL.

Desarrollo web

Tengo experiencia en desarrollo web usando HTML, CSS, JavaScript y React. Creó marcos HTML semánticos, diseñó interfaces responsivas con CSS y desarrolló aplicaciones interactivas con JavaScript. En React, trabajó con componentes funcionales y de clase, ganchos, gestión de estado con Context API y Redux, y bibliotecas como React Router y Axios.
